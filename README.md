# Calculadora de Pegada de Carbono de Viagens

## Sobre

Este é um projeto de uma calculadora de pegada de carbono para viagens. Ele permite que os usuários calculem as emissões de CO₂ de suas viagens com base na distância e no meio de transporte utilizado. O objetivo é conscientizar sobre o impacto ambiental das viagens e incentivar práticas mais sustentáveis.

## Funcionalidades

* Cálculo de emissão de CO₂ para diferentes meios de transporte (carro, ônibus, caminhão, bicicleta).
* Preenchimento automático da distância ao selecionar a cidade de origem e destino.
* Opção de inserir a distância manualmente.
* Tema claro e escuro.
* Dicas para reduzir a pegada de carbono.

## Como usar

1. Clone este repositório: `git clone https://github.com/PedroZef/calculadora_eco2_trip.git`
2. Abra o arquivo `index.html` em seu navegador.
3. Selecione a cidade de origem e destino, ou insira a distância manualmente.
4. Escolha o meio de transporte.
5. Clique em "Calcular Emissão" para ver o resultado.
